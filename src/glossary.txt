Bildanalyse: 
	ist das Berechnen von Bildkomponenten und deren Eigenschaften, z.B. => Kantenerkennung,
	=> Objektlokalisierung, => {Motion Tracking}

Bildverarbeitung: -
Bildverstehen: 
	ist das Berechnen der Bedeutung von Bildern, z.B. => Objekterkennung, => Szeneninterpretation
	und => {Natürlichsprachliche Beschreibung}. Es erinnert an das Verstehen eines Stummfilms:
	es beruht auf Allgemeinwissen und wahrgenommenen Emotionen.

	Bedeutung.
